# Architecture Model Vue presenteur

Architecture Model vue presenteur est un type d'architecture basé sur les models qui sont les entités,
la vue qui va se charger de l'affichage et le présenteur qui est un passe-plat des modèles vers les vues incluant
également les traitements de formatage des données.

## Models

Nos modèles sont:

- Events
- event_creator
- guests
